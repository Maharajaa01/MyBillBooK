# Billing Book App

## Description
A simple mobile app to keep track of products and their prices, designed for use in a small grocery store. This app allows the user to add items, view the total, and manage a basic billing system.

## Features
- Add product name and price.
- View a list of added items.
- Automatically calculates the total amount.
- Simple and user-friendly interface.

## Tech Stack
- React Native
- Expo
- TypeScript (optional)

## Getting Started

### Prerequisites
Before you begin, make sure you have the following installed:
- [Node.js](https://nodejs.org/en/)
- [Expo CLI](https://docs.expo.dev/get-started/installation/)

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/YourUsername/billing-book.git
    ```

2. Install dependencies:
    ```bash
    cd billing-book
    npm install
    ```

3. Run the app:
    ```bash
    npx expo start
    ```

4. Scan the QR code using the Expo Go app (available on Android and iOS) to run the app.
